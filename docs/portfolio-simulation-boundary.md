# Portfolio Simulation Boundary

Portfolio functionality is limited to hypothetical scenario simulation and historical backtesting.

## In scope

- Hypothetical positions and assumptions.
- Scenario-level estimated deltas based on forecasts.
- Backtest summaries for research workflows.
- Clear uncertainty and limitation text.

## Out of scope

- Broker integrations.
- Order placement or execution.
- Portfolio management automation.
- Suitability checks or personalized investment advice.
- Buy/sell signals, recommendations, or target prices.

All UI and API copy must preserve the simulation-only boundary.
